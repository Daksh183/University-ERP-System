# University ERP System

A Java-based **University ERP (Enterprise Resource Planning)** system built using **Core Java, Swing (GUI), JDBC, MySQL, and Maven**.  
The project provides a structured desktop application for managing university-related operations with secure authentication and database connectivity.

---

## Features
- Secure login system (password hashing using BCrypt)
- Desktop GUI built with Java Swing + FlatLaf
- Separate Authentication DB and ERP DB
- Clean layered architecture (UI, Service, Data, Domain)
- Maven-based dependency management
- MySQL database integration

---

## Tech Stack
- **Language:** Java  
- **UI:** Java Swing, FlatLaf  
- **Database:** MySQL  
- **Build Tool:** Maven  
- **Security:** jBCrypt  
- **Charts:** JFreeChart

## How to Run
1. Install **Java JDK 17+**
2. Install **MySQL** and create the required databases
3. Configure DB credentials locally
4. Open the project in **IntelliJ IDEA**
5. Load Maven dependencies
6. Run `Main.java`

(Refer to **How_To_Run.pdf** for detailed steps)

---

## Database Configuration
Database credentials are **not hardcoded for security reasons**.

Update them locally inside:
edu/univ/erp/data/DatabaseConnector.java
